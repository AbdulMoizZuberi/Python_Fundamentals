## 🧠 Key Concepts Covered

### 1. Variables & Data Types

- `type()`  
- Type casting:
  - `int()`
  - `float()`
  - `str()`
- Data structures:
  - `list()`
  - `dict()`

---

### 2. Control Structures

- Conditional statements:
  - `if`
  - `elif`
  - `else`
- Loops:
  - `for loop`
  - `while loop`
- Iteration:
  - `range()`

---

### 3. Functions

- `def`  
- `return`  
- `*args`  

---

### 4. List Comprehensions

```python
[expression for item in iterable if condition]
```
